# Multi-Language Batch File Renamer

This project demonstrates how to solve a real-world file automation problem using two different languages.

## Project Structure
- **/powershell**: Contains the native Windows script (`.ps1`).
- **/javascript**: Contains the cross-platform Node.js script (`.js`).

## How to Run
- To run the PowerShell version: `node powershell/rename_salary_files.ps1`
- To run the JavaScript version: `node javascript/rename_salary_files.js`


I uses these files on .xlsx file type , you can use on your needs
